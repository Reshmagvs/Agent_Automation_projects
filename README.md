# Agent_Automation_projects

This repository is a collection of all my agent development and agent automation projects .

1) [Student Tutor Agent](https://github.com/Reshmagvs/Digi_Shiksha)
AI-driven educational platform designed for students in Indian government schools. It provides personalized, multilingual tutoring in foundational literacy and numeracy .

2) [Google ADK](https://github.com/Reshmagvs/Google_adk_agents)
Various types of agents made using google ADK for stock analysis .

3) [Coffee shop chat assistant](https://github.com/Reshmagvs/BrewBot_workshop) Created a coffee shop chat bot assistant for starbucks using chatling.

4) [Gmail Automator](https://github.com/Reshmagvs/Draft_Gmail_automator) Using n8n , created a AI agent for automation of emails .

5) [AI Chat Assistant](https://github.com/Reshmagvs/Basic_AI_Chat_Assistant_n8n) Uaing n8n created a very basic AI chat assistant.
